** Website Self-Order "The Cakery"
*** Web self-order "The Cakery" adalah sebuah website yang saya dan teman saya buat untuk tugas PKKWU. 
Ini adalah web pemesanan bukan pesan antar, web ini hanya dijalankan di area toko saja, tidak dapat di jalankan secara public.
Web ini untuk memudahkan pemilik toko dalam mengelola stok, laporan keuangan, dan catatan pesanan. 
Web ini dibuat agar pelanggan lebih leluasa dalam memilih produk, meminimalisir kesalahan dalam pencatatan pesanan,dan mempercepat proses penghitungan harga bayar. 
Web ini masih menggunakan sistem pembayaran tunai.
